canine.xargs¶

Contains code to run the xargs orchestrator

class canine.xargs.Xargs(command: str, inputs: Dict[str, Any], backend: Union[canine.backends.base.AbstractSlurmBackend, Dict[str, Any]], name: Optional[str] = None, cwd: Optional[str] = None, resources: Optional[Dict[str, Any]] = None)¶

Simplified Orchestrator Built to work on a local slurm cluster. No localization is performed. Job-specific commands are dumped to a staging directory, and jobs are dispatched. Outputs are not gathered or delocalized, and the only cleanup performed removes the temporary staging dir
